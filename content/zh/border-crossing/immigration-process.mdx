你的航班降落在中国的机场。你走出机舱进入登机桥。头顶上方的中英文指示牌为你指明方向。整个过程遵循严格的顺序。以下是步步详解，包括你所见的、所听的以及需要做的每件事。

## 步骤指南

### 第1步：跟随指示前往 到达/入境

离开机舱门后，廊道分叉。一个牌子写着"中转 / Transfer"向左指。另一个牌子写着"到达 / Arrivals"直行指。你沿着"到达"走。如果你有中国境内联程航班，沿"中转"走——但要知道你仍然在最终目的地清关，而不是中转点，除非你离开机场。

在主要枢纽——北京首都、上海浦东、广州白云——标识会以中文、英文、韩文和日文显示。跟随英文"Arrivals / Immigration"标识。没有走错路的可能——廊道将所有到达旅客汇聚到入境处。你不会不经入境就误走到行李提取处。

从登机口步行至入境处的时间：根据登机口位置不同为5-15分钟。在上海浦东T1，部分远端登机口需要走12分钟外加一段穿梭列车。在北京首都T3，步行需要10-18分钟。不要在洗手间或免税店停留——你的第一站是入境检查。入境检查后方有洗手间和商店。

### 第2步：健康申报二维码扫码

在到达入境大厅之前，你须经过一处置有扫描闸口的检查点。这就是健康申报检查点。一行齐腰高度的扫描仪排列成行，每台带有小屏幕和红色扫描光线。

你将手机对准扫描仪——健康申报二维码朝向红色光线。光线读取二维码。屏幕闪绿光并显示"通过"。你继续前行。

如果你在飞行前没有填写健康申报表格：
- 扫描区域附近设有自助终端，提供英文选项
- 你扫描护照，输入航班号、座位号，并回答3-5个健康问题（过去14天内的症状、过去14天内去过的国家、是否接触过确诊新冠病例）
- 终端打印一张二维码。拿到闸口扫描
- 这会使你的流转时间增加10-15分钟

飞行前截屏保存健康申报二维码。要截屏，因为机场WiFi可能慢、漫游数据可能连不上，你不想成为那个站在检查点前狂刷网页而身后排起队的人。

### 第3步：入境检查——选择你的通道

入境大厅在你面前展开。这是一个大型房间，排列着多行柜台——根据机场规模不同有30到80个柜台。每个柜台区上方的电子标识显示通道类型。仔细看标识。

**你看到的通道类型：**

- **"外国护照" / "Foreign Passports" / "Foreigners"**——你的通道。你走的就这个。找到标有此标识的柜台区。穿着制服的边检人员坐在玻璃隔板后面。

- **"中国公民" / "Chinese Citizens"**——不要在这里排队。甚至不要往这些柜台走。中国公民专用通道在一些机场物理隔离。如果你不小心排了这里的队，柜台官员会把你指引到外国护照通道。

- **"自助通关 / E-Channel"**——供已有指纹登记的生物特征护照持有人使用。如果这是你第一次入境中国，跳过。如果你以前来过中国并成功使用过自助闸口，可以使用这些。详见自助闸口指南。

- **"外交 / APEC / 机组"**——为外交护照持有人、APEC商务旅行卡持有人和机组人员开设的专用通道。除非你持有这些证件，不要在这里排队。

- **"免签"或"24/72/144小时过境"**——部分机场（上海浦东、北京首都、广州白云、成都天府）设有免签和过境免签旅客专用通道。如果有，走这些——队伍通常较短。

排队纪律严格。站在柜台黄线后。等待边检人员示意你靠近。前一位旅客还在柜台时不走上前。

等待时间：5-45分钟，取决于同时有多少架国际航班降落。上午（6:00-9:00）和晚上（5:00-8:00）是国际到达高峰窗口。

### 第4步：在入境柜台——你递交什么

你走到柜台前。一名穿制服的边检人员坐在玻璃后面。你和边检人员之间：一个柜台台面、一条递文件的小口、在你一侧的指纹扫描仪以及边检人员窗口上方对着你的摄像头。

**经文件口递入：**
- 你的护照，翻开至照片页
- 你的入境卡，如果飞机上发放了（大多数主要机场不再使用纸质入境卡——如果你飞行中没有拿到，就不需要）

**入境卡样式（如果航班发放了）：**

```
中华人民共和国出入境边防检查站
ARRIVAL CARD

姓: ____________________
名: ____________________
性别: □ 男  □ 女
出生日期: ____ / ____ / ________ (日/月/年)
国籍: ____________________
护照号码: ____________________
签证号码: ____________________（如适用）
航班号/船名: ____________________
来华目的: □ 旅游  □ 商务  □ 探亲  □ 工作  □ 学习  □ 其他
在华住址: ____________________
签名: ____________________
日期: ____ / ____ / ________
```

用黑色或蓝色笔以大写印刷体填写。在华住址：写酒店名称和城市。"北京希尔顿酒店，北京东方路1号"足够了。

### 第5步：在你等待时边检人员做什么

你递过护照。以下是边检人员按顺序做的事：

1. **打开你的护照至照片页。** 举起来，看照片，看你的脸。再看照片，再看脸。这个直接比对用3-5秒。

2. **扫描护照资料页。** 将你的打开护照在桌面扫描仪前过一下。扫描仪发出蜂鸣声——你的护照机读区（MRZ）数据现在已进入系统。边检人员的屏幕上弹出你的姓名、国籍、出生日期和护照号码。

3. **检查屏幕看是否有警报。** 边检人员盯着显示器看5-15秒。这期间，系统将你的护照与以下数据库进行比对：通缉人员数据库、入境黑名单、过往入境逾期记录和挂失护照登记册。这是你入境过程中最重要的15秒。无论屏幕显示什么，边检人员的面部保持中立。

4. **向系统录入信息。** 如果没有警报弹出，边检人员录入：你的来华目的（从下拉菜单中选择）、计划停留时长、住宿地址和航班号。这些录入用30-60秒。你听到键盘敲击声。

5. **检查你的签证或免签状态。** 对持签证者：核实签证有效、未过期、正确的入境类型（单次或多次）以及允许停留时长。对免签旅客：核实你的国籍符合资格、你本日历年未超过最长停留天数、且你来华目的合理。

6. **可能提问。** 详见下方提问部分。边检人员边打字边提问。他们提问时不停止打字——打字和说话同时进行。

7. **指示你提供指纹。** 边检人员指向你那一侧的指纹扫描仪。参阅下方指纹扫描部分。

8. **给你拍照。** 头顶上方的摄像头在指纹扫描过程中自动抓拍。没有闪光灯、没有倒计时、没有"请微笑"的指示。照片用于入境记录。

9. **在你的护照上盖章。** 边检人员拿出一个章，按在你的签证页或空白页上，并用笔写一个数字。参见下方盖章部分。

10. **递回你的护照。** 边检人员将护照经口递回。要么什么也不说，要么说一句"可以了"。你接过护照往前走。

柜台全程耗时：如果不提问，60-120秒。如果提问并回答完毕，2-5分钟。

### 第6步：指纹扫描——确切顺序

边检人员指向你那一侧的指纹扫描仪。扫描仪是一块玻璃方板，约两张邮票大小，呈轻微倾斜安装，边缘有一圈蓝光。

**扫描顺序：**
- **第一步：** 右手，四指并拢（不含拇指）。将食指、中指、无名指和小指平放在扫描玻璃上。均匀按压——不要太轻也不要太重。注册成功后蓝光变绿光。
- **第二步：** 左手，四指并拢。同样流程。
- **第三步：** 双拇指同时。将两只拇指并排平放在扫描玻璃上。均匀按压。

扫描仪采集你指纹的高分辨率图像。与数据库比对——如已有记录则寻找匹配（如果你以前入境过），如为首次入境则建立新记录。

**如果扫描仪拒绝你的指纹：**
- 你会听到一声低沉的蜂鸣并看到红色闪光。光线恢复蓝色。
- **原因1：手指干燥。** 对指尖哈气增加湿度。手指互相摩擦。再次按压。这能解决80%的失败。
- **原因2：按压力度过轻或过重。** 调整按压力度。想象按下电梯按钮的力度——坚实但不猛烈。
- **原因3：手指没放平。** 每个手指的整个指腹放在玻璃上——不是指尖。
- **原因4：汗液、割伤或疤痕。** 如果某根手指无法扫描，边检人员可能要求你换用另一只手的手指，或者以部分指纹完成。这不是危险信号——有些人就是指纹难以扫描。

边检人员不会烦。指纹失败每班会发生很多次。尝试两次，如果仍然失败，边检人员以现有数据继续。

### 第7步：边检人员可能问的问题——以及他们期望的答案

边检人员的职责是确认你所称目的与你的入境许可匹配，且你无逾期或非法务工风险。他们的问题简短、直接，要求用一句话回答。

**问题1："你来华目的是什么？"**

边检人员边打字边提问，眼睛不离开屏幕。

你的回答必须与你签证或免签类别完全一致。如果你持旅游签证入境却回答"拜访供应商"，你就有麻烦了——商务活动需要商务签证。

- 签证类型：L（旅游）→ 回答："旅游"或"观光"
- 签证类型：M（商务）→ 回答："商务会议"或"拜访客户"
- 签证类型：F（交流/访问）→ 回答："参会"或"学术交流"
- 免签入境 → 回答："旅游"（免签入境仅用于旅游、商务和探亲）
- 过境免签（24/72/144小时）→ 回答："过境。我将飞往[目的地]，日期是[日期]。"

不要展开。不要解释。"旅游"就是一个完整的回答。

**问题2："你打算待多久？"**

说天数，不要说日期范围。不好的回答："大概一两个星期，可能到25号。"好的回答："14天。"你所说的停留天数必须在你的签证或免签许可范围内。如果免签允许30天而你说"六个星期"，你已经宣布了逾期意图。在到达柜台前搞清楚你的最长停留期。

**问题3："你住哪里？"**

说酒店名称和城市。好的回答："北京半岛酒店。"如果还没有订酒店，说城市："我会在上海找酒店。"要有一个说得通的答案。  "我不知道"会引发追问和潜在延误。

如果住在朋友或亲戚家，说："住在[城市]的朋友家。地址是[地址]。"把地址写下来——用中文字符，不是拼音。在手机上或打印的纸上出示地址。

**问题4："你有回程机票吗？"**

回答："有。"如果你手机上有预订确认，主动提出出示。边检人员可能不要求看——问题本身有时是一种筛查技巧。边检人员观察你对问题的反应比检查机票更多。平静的回答加上伸手拿手机的动作就是预期反应。

如果没有回程机票，陈述你的计划离境："我计划订[日期]的机票。"准备被追问。没有回程机票的免签旅客更可能被询问，因为规则技术上要求有前往第三国的证明。

**问题5（针对商务旅客）："你拜访哪家公司？"**

说出公司名称。边检人员输入系统。如果公司名是中文，读出来自业务邀请函——你的接待公司提供的信件。边检人员可能会将公司名称与你的签证申请交叉比对。

**问题6（少见，较长时间停留）："你的职业是什么？"**

用一两个词回答你的职位："工程师"、"教师"、"顾问"、"退休"。这验证你所称的目的——持旅游签证的教师来旅游是合理的。持旅游签证停留29天的"顾问"可能引发追问："你在停留期间工作吗？"正确答案是"不，只是旅游。"

**如何处理语言障碍：**

如果边检人员问的问题你不理解：
- 说："Sorry, could you repeat that?"（对不起，能再说一遍吗？）
- 边检人员换个说法或放慢语速
- 如果仍不清楚，边检人员叫来会说英文的同事。等待时间：主要机场2-10分钟，小机场更久
- 边检人员也可能使用手持翻译设备——正常对着它说话，不要慢也不要大声

不要装作听懂了。回答错误比回答说没听懂造成的问题大得多。

### 第8步：检查你的入境章

边检人员递回你的护照。检查章之前不要走开。检查只需5秒钟。

你的入境章是一枚紫色的矩形章，盖在护照空白页上。它显示：

- **入境日期：** 格式为YYYY MMM DD（例如 2026 JUN 15）。核实日期正确。
- **入境口岸：** 中英文机场代码（例如"北京 PEK"或"上海浦东 PVG"）。
- **准许停留时长：** 由边检人员用笔书写。天数（例如"30"）或日期（例如"2026 JUL 15"）。这是你必须离开中国的截止日期。
- **签证类别代码：** 括号内——(L)表示旅游，(M)表示商务，等。

**核实三件事：**
1. 入境日期为当天日期。如果章显示的是昨天的日期，属于盖章错误——虽罕见但现在可纠正，事后几乎无法修复。
2. 停留时长与你的签证匹配。如果你的免签入境允许30天而章上写的是"15"，边检人员可能写错了数字。礼貌地指着说："Thirty days?"边检人员可以更正。
3. 章清晰可辨。如果墨水晕开导致停留时长无法辨认，你就有麻烦了。请边检人员澄清。

**如果章有错：** 不要离开柜台。指着章说"Sorry, this says [X] days. My visa allows [Y] days."边检人员重新核查系统，划掉错误，写上正确数字并盖一个小更正章在旁边。

如果你带着错误的章离开事后才发现，更正需要前往出入境管理部门——一个耗时数小时的过程。

### 第9步：行李提取

入境检查后，跟随标识到行李提取。标识写着"Baggage Claim / 行李提取"。电子显示屏显示航班号和对应的传送带编号。将你的航班号与传送带编号匹配。

免费行李手推车提供。投入1元硬币释放（一些新型推车免费——推即开）。到海关之后才离开推车——你需要它从行李提取穿过海关区。

等待行李时间：到达传送带后10-30分钟。把这个纳入你的总到达时间。

### 第10步：海关——绿色或红色通道

带着行李进入海关检查区。关于通道选择、免税限额和检查程序的详细说明，参见完整海关指南。概要：无申报物品走绿色通道。有物品或现金需申报走红色通道。

### 第11步：走出到达大厅

海关后，自动玻璃门打开进入公共到达大厅。这是你从飞机到中国境内的整个旅程的终点。

在到达大厅，正前方：
- 举着写有名字的牌子的人群——司机、酒店代表、家属
- 左侧或右侧墙边的货币兑换柜台
- 紧挨货币兑换柜台的ATM机
- SIM卡服务台——中国移动、中国联通、中国电信——穿制服的工作人员
- 大厅中央的问询台
- 指向：出租车、地铁、巴士、停车场的标识

## 常见错误

- **排错入境通道。** 排在中国公民通道里是首次入境最常见的错误。排队20分钟后走到柜台前，边检人员挥手让你走，然后你在正确通道重新排。
- **没有准备好住宿地址。** "我还没订酒店"会触发额外审查。至少在起飞前预订首晚住宿。入境后如果计划有变可以取消。手机上的酒店预订确认函3秒钟就解决这个问题。
- **来华目的说得含糊。** "我就是到处转转，看看东西"太模糊了。"旅游——北京、上海和西安，14天"足够具体。具体性表明旅客有计划，而非可能来找工作的人。
- **指纹扫描不过就慌。** 干燥的机舱空气使指纹难以读取。这发生在约20%的旅客身上。这与你的身份、证件或入境资格毫无关系。尝试对手指哈气再次按压。
- **不看章就走。** 到了酒店发现错误。你现在必须在工作时间亲自前往出入境管理局，去纠正一个在柜台只需30秒就能纠正的入境错误。
- **联程航班混乱。** 如果你入境后还要搭国内联程航班，你需要预留出入境检查的常规45-60分钟，加行李提取15-20分钟，加在国内航站楼重新托运15-20分钟，加换航站楼时间。在国际转国内90分钟联程在主要枢纽几乎不可能完成。
