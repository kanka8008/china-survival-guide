中国城市的地铁系统干净、高效且覆盖面广。截至2026年，中国运营着全球最大的地铁网络——47座城市拥有运营中的地铁系统，运营里程超过12,000公里。支付系统是唯一让外国游客感到棘手的地方——但只要手机上已安装了支付宝或微信，你已经完成了90%的准备。

## 高峰时段：何时避开地铁

高峰时段会将即使最宽敞的中国地铁站变成人挤人的拥挤状态。如果你看重个人空间或携带行李，避开以下时段：

| 城市 | 早高峰 | 晚高峰 |
|---|---|---|
| 北京 | 7:30 — 9:00 | 17:30 — 19:00 |
| 上海 | 7:30 — 9:00 | 17:30 — 19:00 |
| 广州 | 7:30 — 9:00 | 17:30 — 19:00 |
| 深圳 | 8:00 — 9:30 | 18:00 — 19:30 |
| 成都 | 8:00 — 9:00 | 17:30 — 19:00 |
| 重庆 | 7:30 — 9:00 | 17:30 — 19:00 |

高峰期实际情况：列车每2-3分钟一趟，但即刻满员。你可能需要让2-3趟列车过去才能挤上去。北京1号线、2号线、10号线，上海1号线、2号线、8号线，以及广州3号线最为拥堵。高峰期最挤的单个站点：北京**西二旗**站，上午8:30，站台入口排队延伸200米。

**周日出行：**周日地铁系统较为清静。周六中午商业区仍可能繁忙。如果时间灵活，周日上午是最舒适的乘车时段。

## 方法1：支付宝或微信乘车码（推荐）

这是最简单的方法，无需额外下载App。超过90%的中国地铁乘客使用此方式。

![Chinese metro subway guide](/images/articles/metro-subway-guide.png)

### 支付宝乘车码：完整设置步骤

1. **打开支付宝。**首页会看到一个图标网格。"出行"图标位于第一行或第二行，显示一个小火车或公交车图标。如果看不到，点击顶部搜索栏输入"transport"——支付宝的英文搜索可以识别此词。
2. **点击"出行"。**出行页面打开，顶部有两个标签："地铁"和"公交"。选择**地铁**。
3. **城市检测。**支付宝通过GPS检测你所在的城市。城市名称显示在屏幕顶部。如果显示错误的城市，点击城市名称，用英文搜索你当前所在的城市（例如输入"Shanghai"或"Beijing"——两者都可以）。
4. **激活乘车码。**城市名称下方，有一个醒目的按钮："开通/领卡"。点击它。会弹出一个服务条款页面。滚动到底部（"同意"按钮在底部），点击后，乘车二维码在2秒内生成。
5. **支付押金（部分城市）。**北京、上海及其他少数城市过去需要¥20可退还押金。到2025-2026年，大多数城市已取消此要求。如果系统提示缴纳押金，该押金在注销乘车卡时退还。
6. **扫码进站。**在地铁站，走到进站闸机前。在每排闸机上方，寻找扫码器——一个带有二维码图标的矩形玻璃面板。将手机屏幕朝向玻璃，距离扫码器5-10厘米。闸机发出蜂鸣声，在不到半秒内打开。
7. **扫码出站。**在目的地站，在出站闸机处扫描同一个二维码。系统按距离计算票价（¥2-¥10不等）并从你绑定的支付宝支付方式中扣款。一条通知显示具体票价和剩余余额。

### 微信乘车码设置步骤

1. 打开微信，点击**我**（右下角标签），然后点击**服务**。
2. 点击**交通出行**或向下滚动找到"地铁"小程序。
3. 首次使用，微信会请求位置权限。允许它。
4. 选择城市并激活地铁乘车卡——激活流程与支付宝相同。
5. 二维码出现在微信内。在进站和出站闸机处扫码。

**支付宝和微信的关键区别：**
- 支付宝的出行页面更精致，二维码生成更快，英文标注更好
- 微信乘车码有时需要手动刷新——如果闸机没反应，下拉屏幕刷新
- 两者在票价计算和扣款方面完全一致
- 如果其中一个失败（大约每50次扫码中发生1次），立即切换到另一个，不要在闸机前排查故障

**关键事项：**每个城市需要单独激活。在北京激活的乘车码不会授予你使用上海地铁的权限。激活免费，每个城市只需不到10秒。如果你一次行程到访5座城市，在首次使用每座城市的地铁时分别激活乘车码。

## 方法2：城市专属地铁App

每座主要城市都有各自的官方地铁App。如果你想获得App内线路规划等额外功能，这些App很有用：

| 城市 | App名称 | 备注 |
|---|---|---|
| 北京 | 亿通行 (Yitongxing) | 支持外国手机号注册，有英文版 |
| 上海 | Metro大都会 (Metro Daduhui) | 英文界面，支持外国银行卡支付 |
| 广州 | 广州地铁 (Guangzhou Metro) | 有英文版 |
| 深圳 | 深圳地铁 (Shenzhen Metro) | 基础英文支持 |
| 成都 | 成都地铁 (Chengdu Metro) | 中文为主，英文有限 |

所有城市App均支持外国手机号注册，通过支付宝或微信支付集成可使用外国银行卡付款。

**何时用城市App而非支付宝/微信乘车码：**城市App提供每节车厢的实时拥挤数据（上海Metro大都会显示此项）、英文版服务中断提醒，以及含票价预估的线路规划。在停留超过3天的城市下载App。短途停留时，支付宝/微信乘车码已足够。

## 方法3：单程票（现金备选方案）

每个地铁站都有接受现金的自助售票机。当手机没电或乘车码无法使用时，这是你的备选方案。

![Chinese metro subway guide](/images/articles/metro-subway-guide.png)

### 找到英文按钮

售票机排列在车站入口附近，安检通道之前。它们是大型自助终端，通常为银色或蓝色，配有触摸屏。英文语言按钮总是位于以下两个位置之一：

1. **屏幕右上角**——一个小国旗图标（英国国旗或美国星条旗），或彩色按钮上的白色"English"字样。这是北京、上海、广州和深圳最常见的布局。
2. **屏幕左下角**——较少见，出现在成都和重庆部分较旧的机器上。

在进行任何其他操作之前，先点击英文按钮。整个界面立即切换为英文。

### 购买车票

1. **在主屏幕选择"单程票"（Single Journey Ticket）。**忽略储值卡（交通卡）选项——这些需要中国身份证才能注册。
2. **选择目的地。**屏幕显示以下两种界面之一：
   - **地图视图：**一张示意性的地铁路线图。双指缩放。点击目的地站点。票价自动显示。
   - **线路+站点列表：**滚动线路编号，点击你的线路，然后点击目的地站名。
3. **选择数量。**默认为1张票。如需要更多，点击"+"。总票价自动调整。
4. **投入现金。**机器接受：¥1硬币、¥5纸币、¥10纸币、¥20纸币。北京和上海的机器也接受¥50和¥100纸币——寻找标有"纸币"的入钞口。一次投入一张纸币。机器会拒收破损或严重折痕的纸币。
5. **取走车票和找零。**车票落入膝盖附近的取票口。大多数城市使用圆形塑料代币（约扑克筹码大小）。上海使用非接触式卡片。深圳使用绿色代币。北京使用深蓝色代币。立即取走——机器在15秒后收回未被取走的代币。
6. **在进站闸机处感应。**将代币放在闸机的圆形读卡器上。闸机打开。保管好代币。出站时需要它。
7. **出站。**在目的地站，将代币投入出站闸机的回收口（代币制系统）或在读卡器上感应（刷卡制系统）。闸机吞入并保留代币。如果闸机无法打开，走到车站服务台——工作人员会检查你的代币并放你通行。

## 安检：需要注意什么

中国每个地铁站都有强制性安检。流程很快（非高峰15-30秒，高峰时段1-3分钟），所有城市保持一致。

**安检设置：**
- 对行李和大件物品进行X光传送带检查
- 一道步行通过的金属探测门
- 2-4名身穿深色制服、佩戴"安检"臂章的安检人员

**安检流程：**
1. 走近安检通道。取下背包、单肩包或任何大于手袋的包。
2. 将包放在传送带上。传送带自动运行——无需推任何东西。
3. 步行通过金属探测门。不要在门中间停留。保持移动。探测器很少对手表、皮带或手机发出报警。
4. 在另一端取走你的包。包在5-10秒内出来。

**什么会被检查：**
- 超过50毫升的瓶装液体。安检人员可能要求你喝一口以证明不是易燃品。这种情况频繁发生——将水瓶拿在手里，不要塞在包里。被要求时喝一口然后继续走。
- 充电宝和大容量电池包。安检人员可能把你的包拉到一边检查充电宝的容量标签。超过20,000毫安时的充电宝有时会被禁止携带。
- 气雾罐（除臭剂、防晒喷雾）。这些会触发二次检查。将它们放在方便取出的位置。
- 折叠刀、刀片超过6厘米的剪刀，以及任何类似武器的工具——这些会被没收。

**实用技巧：**将手机拿在手里，不要放在包里。安检后20秒你就要在闸机处扫码。如果手机塞在包里，意味着你得在排队人群的注视下从包里掏出手机。

## 看懂地铁路线图

中国地铁路线图使用示意性（非地理比例）布局，风格与伦敦地铁或纽约地铁图完全一致。理解它们只需要30秒的适应时间。

**线路识别：**
- 每条线路有独特的颜色和编号。每座城市的1号线始终是红色。2号线始终是绿色或蓝色色调。
- 线路编号出现在路线沿线彩色圆圈内。
- 快速线路（跳站不停车的线路）用更粗的路线线条和更少的站点圆点标示。

**方向：**
- 方向使用**终点站名称**标示，而非指南针方向。一块写着"往天安门东"的标识牌意思是"toward Tiananmen East"——列车正驶向名为天安门东的终点站。你需要知道你行进方向上最后一站的名称。
- 在站台标识上，当前站会被高亮显示。终点站出现在线路图两端。

**换乘：**
- 换乘站以较大的圆圈或两条以上线路颜色交汇的重叠圆圈标示。
- 主要换乘枢纽：人民广场（上海，1/2/8号线）、西单（北京，1/4号线）、珠江新城（广州，3/5号线）、市民中心（深圳，2/4号线）。

**票价：**
- 大多数城市前4-6公里收费¥2-¥3，之后每增加10公里加收¥1。在北京或上海，一次贯穿全城的行程票价很少超过¥10。
- 部分城市设有月度封顶：北京刷卡用户封顶¥100，上海封顶¥70。

## 车站标识：你会看到什么

中国地铁站使用统一的双语标识。以下是从入口到站台你会遇到的内容：

| 中文标识 | 英文 | 在何处看到 |
|---|---|---|
| 地铁 | Metro / Subway | 车站入口、地面层指示牌 |
| 入口 / 出口 | Entrance / Exit | 车站进出口和出站闸机 |
| 安检 | Security Check | 安检通道区域 |
| 售票 / 自助售票 | Tickets / Self-Service Tickets | 售票机区域 |
| 乘车 | Board / To Trains | 通往站台的方向指示牌 |
| 换乘 | Transfer / Interchange | 指引你前往换乘线路的指示牌 |
| 开往 [站名] | To [station name] | 站台上显示列车方向的指示牌 |
| 卫生间 | Toilet / Restroom | 卫生间位置指示牌 |
| 无障碍电梯 | Accessible Elevator | 供轮椅使用者和携带行李者使用的电梯 |
| 客服中心 | Service Center / Customer Service | 付费区内有工作人员值守的服务柜台 |

**读懂站台显示屏：**
每个站台上方，数字显示屏显示：
- 线路编号和颜色（例如，红色横条上的"Line 1"）
- 中英文终点站名称
- 到站倒计时："2 min"/"5 min"——列车根据线路和时段每2-8分钟运行一班
- 较新系统（上海、深圳、北京6/7/14/16/17号线）：每节车厢的拥挤程度用彩色条形图显示——绿色（空）、黄色（中等）、红色（满）

## 主要车站换乘导航

### 人民广场（上海）——1号线、2号线、8号线

人民广场是上海最繁忙的换乘站，日客流量120万人次。三条线路在此交汇，分三层地下空间。

- **1号线站台：**地下二层。红色线路，南北走向。
- **2号线站台：**地下三层。绿色线路，东西走向。
- **8号线站台：**地下一层。蓝色线路，南北走向。

**换乘路线：**
- **1号线换2号线：**从1号线站台出发，跟随标有"Line 2 / 换乘2号线"（绿色）的指示牌。乘自动扶梯上一层。通道需要走4-6分钟——这是上海最长的地下步行通道之一。除非别无选择，不要在高峰期在人民广场进行1号线和2号线之间的换乘。通道会变成单列的人流长河。
- **1号线换8号线：**跟随标有"Line 8"的指示牌。换乘较短——通过一条直达通道，2-3分钟。
- **2号线换8号线：**更短——只差一层，2分钟。

**人民广场出站策略：**该站有20个出口，编号1-20。每个出口通往人民广场和周边街道的不同角落。出站前，查看闸机旁的黄色出站导览图。1号出口通往南京东路，9-10号出口通往上海博物馆。选错出口会多走10-15分钟的地面路程，穿过上海最大的十字路口之一。

### 西单（北京）——1号线、4号线

西单是北京的商业中心换乘枢纽。1号线（东西走向）和4号线（南北走向）在此交叉。

- **1号线站台：**深层（地下二层）。较老的线路，站台较窄。
- **4号线站台：**较浅层（地下一层）。较新，站台更宽，照明更亮。

**换乘：**从1号线出发，走到站台中央，乘自动扶梯上一层。跟随标有"Line 4 / 换乘4号线"的头顶指示牌。通道很短——2分钟。从4号线换1号线，方向相反。1号线站台在下一层。

**西单出站提醒：**F1和F2出口通往西单商圈。E出口通往西单北街。出口清晰标示编号和附近地标。该站共有8个出口。

### 通用换乘规则

- **跟随彩色线路指示带。**换乘通道使用与目标线路颜色匹配的彩色地面贴条。如果要换乘2号线（绿色），跟着绿色贴条走。
- **预留换乘时间。**在人民广场、西直门（北京）、南京东路以及类似的大型车站，仅换乘步行就需要预留5-8分钟。较小车站需1-3分钟。
- **换乘站的末班车。**换乘站最后一班衔接列车的时间不一定与始发线路的末班车时间相同。换乘通道张贴的指示牌显示每条衔接线路的末班车时间。如果在晚上10:30之后换乘，请查看这些时间。
- **不要走出付费区。**换乘在闸机内完成。在两条线路之间换乘时不需要扫码或投入代币。如果不小心走出了付费区闸机，需要支付新的行程费用才能再次进站。

## 丢了车票怎么办

### 丢失单程票代币

如果你在列车或站台上掉了代币并且找不到了：

1. **前往出站闸机附近的客服中心。**这是一个有工作人员值守的柜台，通常配有玻璃墙，上面有中英文标识。
2. **说明情况。**主要城市站点的工作人员会讲基础英语。说"I lost my ticket"或在手机上展示翻译："我丢了车票"（wǒ diū le chē piào）。
3. **支付最高票价。**你需要支付该地铁线路的最高票价——在北京和上海通常¥9-¥14。工作人员会开具一张替代出站票。
4. **保留收据。**如果后来找到了原来的代币，有时可以在同一车站的客服中心申请退款。成功率约50%。

### 手机没电 / 乘车码无法工作

如果在行程中途手机电池耗尽：

1. 前往目的地站的客服中心。
2. 告知工作人员你的起点站。他们可以在系统中查询票价。
3. 用现金支付票价。客服中心始终接受现金。
4. 工作人员会手动放你通过出站闸机。

### 丢失交通卡（储值卡）

如果你购买了实体储值卡（不建议短期停留者购买）：

1. 立即在任何客服中心挂失。
2. 提供护照号码。工作人员可以查询并冻结卡片。

## 无障碍设施

自2018年以来，中国地铁系统的无障碍设施已有显著改善。以下是截至2026年的现状：

**电梯：**
- 2015年后建成的每座车站至少有一部从地面到站台的电梯。2015年前建成的车站正在进行改造——约85%的北京车站和90%的上海车站现已配备电梯通道。
- 寻找轮椅符号和"无障碍电梯"标识。这些电梯通常靠近A出口或最靠近街道拐角的出口。
- 在地面层，电梯入口以蓝色指示牌标示，上面有轮椅图标。有时会比主楼梯入口后退10-20米。

**站台无障碍设施：**
- 站台与列车之间的间隙：根据车站年龄，5-15厘米不等。较新车站（北京14号线及以上、上海12号线及以上）近乎齐平登车。
- 触觉铺道（黄色凸起地砖）在所有车站内从电梯延伸至站台边缘。
- 爱心座椅在每节车厢末端以不同颜色标示（通常为橙色或黄色）。

**残障人士服务：**
- 北京地铁：拨打010-96165，提前24小时，可在任何车站申请工作人员协助。工作人员会在入口处接你并护送到站台。
- 上海地铁：拨打021-64370000申请无障碍协助。主要车站可当日提出请求。
- 深圳地铁：在客服中心即时应答协助，无需提前预约。

**卫生间：**
- 约70%的地铁站设有卫生间。它们始终位于付费区外（进站前或出站后）。没有卫生间的车站通常是北京和上海1号线和2号线上的较旧站点。
- 卫生间标识："卫生间"或"洗手间"。免费。自带厕纸和洗手液——这两样常常是空的。
- 所有配有卫生间的车站均设有无障碍卫生间。

## 常见陷阱

- **漏刷出站码。**进出站都必须扫描乘车码。如果出站时忘记扫描，系统按最高票价扣费并记录为"不完整行程"。30天内出现3次不完整行程可能会导致乘车码被暂停使用24小时。如果离开车站后意识到忘记扫码，返回任意闸机处重新扫描——通常可以重新正确处理。
- **城市激活错误。**如果乘车码在闸机处不工作，检查是否已为当前城市激活。支付宝会自动检测位置，但GPS延迟有时会停留在上一座城市。在出行页面手动切换城市。在到达闸机之前完成此项操作——你身后的人毫无耐心。
- **二维码过期。**为了安全，乘车二维码每60秒刷新一次。如果闸机没反应，二维码可能在扫码中途过期。等待2秒自动刷新，或下拉屏幕手动刷新，然后重新扫码。
- **1号线/2号线混淆。**北京1号线、上海1号线和广州1号线是不同城市中完全不同的线路。查询路线时，确认你是在正确城市的地铁系统内。为上海1号线规划的路线在北京毫无意义。
- **末班车时间。**地铁系统在晚上10:30至午夜之间关闭，取决于城市和线路。每条线路的末班车时间在站台屏幕上显示并通过广播播报。如果错过末班车，网约车（滴滴）是你唯一的选择——预计收车时间段的加价幅度达到1.5-3倍。
- **站台两侧开门。**部分车站左侧开门，部分右侧开门。站台安全门上方的电视屏幕在列车到达前以箭头显示开门侧。观察箭头，或直接跟随人群。
- **禁止饮食。**大多数城市禁止在列车和付费区内饮食（上海对违规者处以¥50起的罚款）。饮水通常被默许。热食、奶茶和任何气味浓烈的物品会引来工作人员注意。

## 分步指南

1. **检查时间。**如果在工作日的7:30-9:00或17:30-19:00之间，预期会非常拥挤。多预留10-15分钟用于安检排队和等待能上去的列车。
2. **打开支付宝或微信**，激活当前城市的地铁乘车码。确认城市名称与你所在地一致。
3. **规划路线。**使用Apple Maps、Google Maps或高德地图（高德地图）。记下线路编号、方向（终点站名称）、所有换乘站以及目的地站名。如果数据连接不稳定，截屏保存路线。
4. **进站。**在地面层跟随蓝色地铁logo指示牌。走楼梯、自动扶梯或电梯下楼。
5. **通过安检。**将行李放在传送带上。步行通过金属探测门。在另一端取回行李。非高峰时段全程30秒。
6. **扫码进站。**将乘车码距离进站闸机扫码器5-10厘米。等待蜂鸣声和闸机打开。直接通过——不要犹豫，闸机2秒后关闭。
7. **跟随指示牌前往站台。**下到站台之前，检查头顶指示牌上的线路编号、颜色和方向（终点站名称）。走错站台=浪费5-10分钟。
8. **上车。**站在黄色安全线后。让下车乘客先出再上。爱心座椅（以橙色/黄色标示）应让给老年人、孕妇和残障乘客。
9. **如需换乘。**跟随换乘指示牌（换乘+线路编号）。留在付费区内。跟随与目标线路颜色匹配的彩色地面贴条。
10. **出站。**在出站闸机处扫描乘车码或投入代币。在出站闸机区域稍作停留，查看黄色出站导览图——注意哪个出口字母（A、B、C、D...）离你的目的地最近，然后再走出去。选错出口可能会让你站在一条10车道马路对面，且没有人行横道。
