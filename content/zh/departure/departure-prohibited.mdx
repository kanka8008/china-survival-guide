### 哪些物品完全禁止带出中国？

出境违禁品清单基本与入境禁止清单一致，再加上几类与文化遗产和国家安全相关的：

1. **入境禁止清单中的所有物品**——武器、弹药、假币、毒品、有毒物质以及被认定危害国家利益的物品
2. **涉密文件和材料**——任何含有国家秘密或敏感信息的内容
3. **受保护的文物**——需要出境许可的古董和文物
4. **濒危野生动植物及其制品**——象牙、犀牛角、虎骨、部分传统药材、珊瑚以及来自受保护物种的制品
5. **濒危植物和种子**——受CITES（濒危野生动植物种国际贸易公约）保护的某些植物物种、种子及其衍生物
6. **未获授权的有价值文化物品**——文化遗产管理局已指定为受保护的珍本书籍、手稿、美术品和历史文献

### 违禁品海报展示的内容

在每个中国机场安检之前的通道，你会看到大幅海报或数字屏列出违禁品。海报通常展示：
- 带红色X标记的违禁品照片
- 限制物品的限额和门槛
- 双语文字（中英文）
- 关于野生动植物制品刑事处罚的提示

**海报上列出的标准限制限额：**

| 物品 | 限额 | 备注 |
|------|------|------|
| **酒类** | **2瓶**（总量不超过1.5升）| 涵盖所有含酒精饮料。中国白酒、红酒、啤酒——全部计入同一2瓶/1.5升限额。 |
| **香烟** | **400支**（2条）| 或雪茄100支，或烟草500克。这些配额不能混用。 |
| **人民币现金** | **20,000元** | 实体纸币。低于此额无需申报。 |
| **外币** | **折合5,000美元** | 无需申报。5,000-10,000美元须申报单和银行凭证。 |
| **金银** | **50克** | 超过此需要中国人民银行许可。 |
| **茶叶** | 合理自用 | 大宗商业批量触发出口要求。 |
| **丝绸** | 合理自用 | 同茶叶——个人用量没问题；大宗商业批量需有文件。 |

### 出境携带现金的限额是多少？

出境与入境适用相同规则：
- **人民币：** 实体现金最高20,000元
- **外币：** 折合5,000美元以下无需申报；5,000-10,000美元须申报和银行凭证；超过10,000美元须外汇管理局许可

如果你将现金带入中国并在入境时申报（有盖章申报单），出境时出示该单以说明你要带出的现金。没有它，该笔现金将在出境时被视为未申报。

### 能否将古董和艺术品带出中国？

**取决于物品年代和类别。**

- **1949年前（中华人民共和国成立前）制造的物品**可能被认定为文物。带出中国需持有国家文物局或其授权地方机构颁发的**文物出境许可证**。
- **1949年或以后制造的批量生产且无特殊历史意义的物品**一般可无证出境。
- **知名艺术家的原创画作、书法和雕塑**可能不论年代都需出境许可，如果该艺术家具有文化重要性的尤其如此。
- **从持牌古董店购买的已经加盖红色官方出境章的物品**可以出境——外国人购买古董的最安全方式即此。

### 文物出境许可证

当你从持牌经销商那里购买一件符合出境条件的古董时，该店铺将一枚**火漆印**直接加盖在物品上。这是一枚硬化火漆的小型圆形红色印戳，表明该物品已通过文物审查。无法在不损坏印章的情况下取下来。在机场，海关官员寻找这枚章。没有印章的古董将受到查验且可能被没收。

**独立取得出境证书（不通过经销商）：**
1. 将物品带到当地**文物局**进行评估。
2. 提交申请表、你的护照和购买凭证。
3. 支付鉴定费（按物品价值浮动）。
4. 如获批，你将收到**文物出境许可证**——一份带有物品描述、你的姓名和公章的打字证书。
5. 办理流程需1-2周。务必在离境前很久就办好。

如果你买了一件古董不确定出境资格，与卖方核实。持牌古董经销商知道哪些物品需要许可。无牌街头摊贩可能售卖无法合法出境的物品——机场被扣的风险是你自己的，不是卖家的。

### 如果海关发现违禁品会怎样

海关官员在安检中标记你的行李后，发生的是：

1. 官员将你的行李拉出，要求你打开。
2. 他们识别出违禁物品并说明违规行为。
3. 根据物品不同：
   - **轻微限制物品（略微超过香烟/酒类限额）：** 超额部分没收。收到一份没收通知单。无进一步处罚。
   - **未经出境许可的古董：** 物品没收。你可能被询问购买地点。首次过失性违规不罚款，但物品拿不回来了。
   - **野生动植物制品（象牙、犀牛角等）：** 物品立即没收。你被留下接受问询。可能报警。这是刑事犯罪。预期正式调查、可能被起诉及严厉处罚。
   - **大额未申报现金：** 超额现金被扣押。你必须提供现金来源证明。未申报的折合10,000美元以上且无许可，触发正式海关调查。
   - **敏感材料（政治敏感文件、数据）：** 设备和文件被扣押。你被留下做进一步调查。结果取决于材料性质。

4. 所有情形下，你都会收到一份记载了查扣事项的正式海关通知。你签名。这一通知存入海关系统。
5. **如果检查耗时很长，你会误机。** 海关不会为了让你赶航班而拖延检查。程序排在第一优先。

### 野生动植物制品呢？

近年来，中国海关对野生动植物制品的执法明显趋严，与CITES国际公约更紧密接轨。以下为高风险物品：

- **象牙和象牙制品**——接近全面禁止。甚至是在国外购买、途经中国的老旧象牙制品也可能被扣押。
- **犀牛角**——绝对禁止。
- **虎骨、豹骨、熊胆制品**——禁止。
- **部分传统中药材**——含有麝香、高鼻羚羊角、穿山甲鳞片及其他受保护物种衍生物的制品在国际贸易中被禁止。
- **珊瑚、部分贝壳及来自受保护物种的海产品。**
- **活体动植物**——一般需 CITES 许可证和检疫证书。

如果你带着第三国购买的野生动植物制品经过中国中转，这些规则同样适用。物品在你持有下进入中国领土，适用中国法律。

### 茶叶、丝绸及常见纪念品呢？

自用购买的常见消费品——茶叶、真丝围巾、瓷器、筷子、服装、电子产品、化妆品——只要数量在合理自用范围内且不在禁止清单上，可以不受限制带出。

同款大批量（例如50条同款真丝围巾）可能被作为商业出口对待，有不同海关要求。

### 携带违禁品出境的处罚是什么？

- **物品在安检或海关检查点被没收**——不予经济补偿
- **罚款**——最高可至物品价值的数倍
- **刑事指控**——对于濒危野生动植物制品，严重案件引发刑事追诉
- **误机**——检查耗时，你可能被滞留直至问题解决
- **违规记录**——标记在海关系统中，影响未来出入境

### 如何确认某件物品是否安全可带出

1. **购买时询问卖方**——持牌零售商了解其产品的出口规则
2. **检查物品上是否有出境章**——正规古董店施以火漆印表示该物品已被批准出境
3. **心存疑问就别买**——被没收的经济损失和在海关被拦下的压力超过了有问题的纪念品的价值
4. **购买任何含有野生动植物成分的产品前，查一下CITES名录**

### 笔记本电脑、硬盘和数据呢？

海关官员有权在边境检查电子设备及其内容。虽不是针对每位旅客的常态，但仍可能发生。被认定为涉密、政治敏感或违反中国法律的数据可能导致设备被扣留或旅客被留下接受问询。

这对普通游客极少发生，但旅客应了解中国边境检查权力包含对数字设备的查验。

### 离境前行李快速自查清单

离开酒店前，打开你的行李检查：

- [ ] 无象牙、犀牛角或任何野生动植物制品
- [ ] 无未带火漆出口章的古董
- [ ] 实体现金不超过20,000元人民币
- [ ] 外币折合不超过5,000美元（无申报情况下）
- [ ] 酒类不超过2瓶/1.5升
- [ ] 香烟不超过400支/2条
- [ ] 无涉密文件或敏感材料
- [ ] 所有充电宝在随身行李而非托运行李中
- [ ] 所有液体容器在100ml及以下
- [ ] 手提、平板和电子产品便于安检取出
