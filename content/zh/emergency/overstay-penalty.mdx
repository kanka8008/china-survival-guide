### 在中国逾期停留的处罚是什么？

中国对逾期停留按违规时长分级处罚。以下基于国家移民管理局处罚裁量标准：

| 逾期时长 | 处罚 |
|----------|------|
| 不足10天 | **警告**——记入系统的正式警告。不罚款，但记录永久保留。 |
| 10至60天 | **每天500元**，最高**合计10,000元**，或最高**5日拘留** |
| 超过60天 | **10,000元罚款**或**5-15日拘留** |
| 情节严重（含屡次违规、蓄意规避或拒交罚款） | **10,000元罚款 + 5-15日拘留 + 强制离境 + 入境禁令** |

罚款按非法居留天数计算。"情节严重"是法律术语，包含：反复逾期、试图隐藏逾期、极长逾期、逾期期间非法务工或曾被遣返记录。

### 罚款如何计算——真实案例

罚款从第10天开始逐日累计。以下是你确切需要支付的：

**案例1：逾期5天**
- 5天属"不足10天"类
- 处罚：记入系统的警告。0元。
- 未来影响：短期逾期单次警告很少堵死未来签证，但移民官员可永久看到。

**案例2：逾期15天**
- 15天属"10至60天"类
- 计算：15天 × 500元 = 7,500元
- 或：最高5日拘留（警官裁量）
- 操作中，若主动自报并交款，警官通常从轻采取罚款而非拘留。

**案例3：逾期25天**
- 25天属"10至60天"类
- 计算：25天 × 500元 = 12,500元，但受10,000元上限封顶
- 支付10,000元。

**案例4：逾期65天**
- 65天属"超过60天"类
- 处罚：10,000元罚款 + 5-15日拘留
- 触发入境禁令（见下表）。

**案例5：逾期90天、二次违规**
- 类型：情节严重（屡次违规 + 长逾期）
- 处罚：10,000元罚款 + 15日拘留 + 自行承担费用强制离境 + 5年入境禁令
- 你自行购买遣返机票。由移民警察护送至登机口。

### 出入境章与逾期如何体现

你入境中国时，移民警察在你的护照上盖入境章。该章包含：
- 入境日期
- 入境口岸（如"北京首都"）
- 手写批注："停留至"后跟许可的最后一日
- 小序列号

出境时，警官盖出境章。如果逾期，出境章上会有标注显示逾期。警官将逾期信息录入关联你护照号码的移民数据库。即便实体章上没有明显逾期标记，电子记录也即时更新。未来移民警察扫描你的护照时即看到该记录。

### 问询室内部发生什么

你如果在出入境管理局主动自报逾期，警官带你到一间问询室。是一间小办公室，通常配有：
- 一台带电脑的桌子
- 两把椅子（你和警官各一）
- 一台录像的摄像头
- 如需翻译则有一部电话

警官会问标准问题：
1. 你什么时候入境中国？
2. 你的许留期应何时结束？
3. 你为何未在截止日前离境或延期？
4. 你在哪里住宿的？
5. 你的职业是什么？
6. 你之前逾期过吗？

如实作答。撒谎加重处罚。主动自报已经是减轻因素。不要用不实陈述破坏它。

问询结束后，警官打印一份《行政处罚决定书》。你签字。该文书注明：
- 违规事由（逾期）
- 时长
- 处罚（罚款、拘留或离境令）
- 缴费期限
- 你陈述辩解的权利（虽无正式上诉程序）

随后去缴费窗口，交完罚款，拿到收据。如果被令离境，警官在你的护照上盖有离境截止日期的章。

### 入境禁令时长表

| 逾期严重程度 | 禁令时长 |
|-------------|---------|
| 不足10天、首次违规 | 不设禁令（仅警告） |
| 10-30天 | 6个月至1年（警官裁量） |
| 31-60天 | 1-3年 |
| 61-90天 | 3-5年 |
| 超过90天 | 5年至永久 |
| 屡次违规（任意时长） | 自动最低5年 |
| 逾期期间非法务工 | 5年至永久 + 罚款 |

这些禁令为正式法律令，存在于中国出入境系统及所有边检站。未来签证申请中隐瞒禁令构成欺诈，触发永久禁令。

### 如何申请特殊减免

在极端情形下可请求从宽。这并非正式"上诉"——而是请求警官考虑减轻因素。

**有效从宽理由：**
- 住院治疗导致无法旅行（提交住院记录、出院小结、医生证明称不适合旅行）
- 因自然灾害或特别政府行为导致航班取消（提交正式取消通知、使领馆声明）
- 丧失行动能力（提交医疗记录证明确实无法早日报到）

**无效理由（当场驳回）：**
- "我忘了日期"
- "我想多看些地方"
- "朋友婚礼在我的截止日期之后"
- "航空公司网站太复杂"

### 逾期处罚会影响未来来华吗？

**会影响。** 逾期记录在出入境系统中永久关联你的护照号码。后果：
1. **未来签证申请。** 你的签证申请会被标记。可能被要求解释逾期事由。批准无保证。
2. **未来免签入境。** 边境移民警察能看到逾期记录。即使你的国籍符合免签资格，也可能被拒绝入境。
3. **入境禁令。** 对于严重逾期（60天+），出入境管理局可施加**1至5年**的正式入境禁令，极端情形下永久禁令。

### 如果我已经逾期了应该怎么办？

**立即去出入境管理局。** 不要试图隐藏逾期或指望没人发现。每拖延一天都加重处罚。

到那之后：
1. 如实说明情况。在被发现前主动自报视为减轻因素。
2. 携带护照、住宿登记回执和任何能解释逾期原因的文件（航班取消通知、医疗记录等）。
3. 你会接受警官问询。他们基于时长和情节确定处罚档级。
4. 如需则交罚款。办公室提供收据和正式处罚通知。
5. 如果被令在指定日期前离境，严格遵守。处罚通知上的离境截止日没有商量余地。

### 如果我因被查出而非自报被发现逾期怎么办？

处罚显著加重。通过警察检查、酒店登记或边检站被发现：
- 取消自报的减轻效果
- 增加拘留可能性
- 入境禁令概率上升
- 可能导致被押送至机场立即离境

### 导致逾期的常见情形

- **天数计算方法搞错。** 30天免签从到达次日零时开始计算。你6月1日到达，第1天是6月2日，第30天是7月1日。你必须在7月1日前离境。部分旅客将到达日算作第1天，导致计划离境日晚了一天。
- **航班延误。** 你停留最后一天晚上10点的航班被取消并改至次日早晨。你现在技术上逾期了。操作中，有记录的航班取消和立即自报通常能从宽，但不确保。
- **没看入境章。** 部分旅客不看移民警察给的章。如果警察写了比你预期短的停留期（如15天而非30天），以盖章数字为准——不是你理解的政策那样算。
- **从签证签发日而非入境日计算。** 你签证的有效期（如"请于12月31日前入境"）是入境中国的时间窗口。你每次停留时长（如"每次停留：30天"）从入境章日期开始计算。混淆两者导致错误计算。

### 如何避免逾期

1. **在入境窗口立即拍照你的入境章。**
2. **仔细计算许可停留天数**——从到达次日零时起，而非到达时刻。
3. **在停留到期前7天设置手机提醒。** 这给你时间申请延期（如需要），或调整离境计划。
4. **如果返程在最后一天**，订早上的航班——不要推到傍晚。傍晚航班被取消，你没有缓冲。
5. **如果知道需要更多时间，提早申请延期。** 不要等到最后48小时。
