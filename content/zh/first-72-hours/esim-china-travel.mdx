eSIM是嵌入手机的数字SIM卡。对于中国旅行，它解决两个问题：落地即刻联网，且免去办理实体SIM卡需要的护照登记环节。

## eSIM vs 实体SIM：你需要放弃什么

在选择eSIM之前，先理解这个取舍。大多数中国旅行eSIM为**纯数据**服务——提供互联网接入但没有中国手机号码。

| 特性 | eSIM | 实体机场SIM卡 |
|------|------|-------------|
| 中国手机号码 | 通常没有 | 有 |
| 数据速度 | 4G/5G | 4G/5G |
| 护照登记 | 不需要 | 需要 |
| 激活 | 出发前 | 机场柜台 |
| 语音通话 | 仅通过App（无原生通话） | 有，含国内通话 |
| 短信 | 无 | 有 |
| 价格（7天/5GB） | 8-20美元 | 50-80元（7-11美元） |

中国手机号码在以下场景是必需的：
- 部分App注册（美团外卖、某些打车账户）
- 接收短信验证码
- 拨打本地电话联系餐厅、酒店或服务号码

如果你只需要地图、翻译、社交媒体和网页浏览的数据，那么纯数据eSIM够用了。如果需要电话号码，请在机场办理实体SIM卡。

### 哪个供应商适合你

**选Airalo**如果你看重可靠性和轻量App体验。App实时追踪数据用量并在接近限额时发送通知。充值即时到账。界面清晰流畅。对首次使用eSIM的用户来说，这是最稳妥的选择。

## 截屏指南——iPhone端eSIM设置

1. **打开 设置 → 蜂窝网络（或移动数据）。** 在较新iPhone上，这是"设置"中顶部区域，紧挨Apple ID下方。

2. **点击"添加eSIM"。** 该选项出现在当前蜂窝套餐下方。如果没有看到，你的iPhone型号或运营商可能不支持eSIM。在 设置 → 通用 → 关于本机 中——查找"EID"（一个32位数字）。没有EID就表示不支持eSIM。

3. **出现"设置蜂窝网络"屏幕。** 两个选项："从附近iPhone传输"和"使用二维码"。选择"使用二维码"。

4. **扫描二维码。** 将iPhone摄像头对准供应商邮件中的二维码。将手机平稳保持在约15-20厘米距离。顶部出现通知横幅："蜂窝套餐已准备好安装"。点击。

5. **在确认屏幕上点"继续"。** iPhone向你展示套餐详情：供应商名称、套餐类型、数据量。核实与你购买的一致。

6. **为套餐命名。** iPhone提示你为这一新线路做标签。从建议中选"旅行"或输入"中国eSIM"。此标签显示在"设置"中，帮你区分与本国号码。

7. **选择默认线路行为。** 将你的本国号码设为语音/短信的"主号"，新eSIM设为"副号"或"仅蜂窝数据"。之后可以更改。

8. **将eSIM设为数据线路。** 设置 → 蜂窝网络 → 蜂窝数据 → 选择"中国eSIM"（或你的自定义标签）。这会将所有互联网流量通过eSIM路由。

9. **为eSIM线路打开数据漫游。** 设置 → 蜂窝网络 → 点击eSIM线路 → 将"数据漫游"开关打开。这是人们常忘记的一步。不打开，eSIM就不会连接，因为它技术上是在中国联通网上漫游。

10. **落地前保持eSIM关闭。** 设置 → 蜂窝网络 → 点击eSIM线路 → 将"启用此号码"开关关闭。只有当你的飞机落地中国时再启用。

## 截屏指南——Android端eSIM设置

注意：Android界面因制造商而异。以下步骤适用于原生Android（Google Pixel、摩托罗拉），与三星One UI也接近。

1. **打开 设置 → 网络和互联网 → SIM卡。** 三星上：设置 → 连接 → SIM卡管理器。

2. **点击"+"图标或"添加eSIM"。** 部分手机称之为"改为下载SIM卡"或"添加移动套餐"。

3. **扫描二维码。** 将手机对准二维码。Android通常自动检测并显示"发现移动套餐"提示。点击"下载"或"继续"。

4. **为eSIM命名。** Android要求输入SIM卡名称。填写"中国eSIM"，以便在双卡设置中识别。

5. **设置数据首选项。** 在"移动数据首选SIM"下选择新eSIM。在"通话和短信首选SIM"下选择你的本国SIM。

6. **为eSIM打开数据漫游。** 设置 → 网络和互联网 → SIM卡 → 点击eSIM → 将"漫游"开关打开。三星上：连接 → SIM卡管理器 → 点击eSIM → 将"数据漫游"开关打开。

7. **到达前禁用eSIM。** 设置 → 网络和互联网 → SIM卡 → 将eSIM开关关闭。

## APN设置——何时需要手动配置

95%的用户永远不需要碰APN设置。eSIM配置文件含有正确的接入点名称（APN），手机自动配置完成。但如果在排除了其他故障步骤后仍然连不上，检查APN：

**iPhone上：** 设置 → 蜂窝网络 → 点击eSIM线路 → 蜂窝数据网络 → APN字段。供应商的邮件/文档中包含了正确的APN值（基于中国联通的eSIM通常为"3gnet"）。准确输入。不要改动这个屏幕上的任何其他内容。

**Android上：** 设置 → 网络和互联网 → SIM卡 → 点击eSIM → 接入点名称。如果列表为空或者选中的APN看起来不对，点击"+"添加新的。输入供应商文档中的APN名称和值。保存。选择新APN。重启eSIM线路（开关关闭再打开）。

## 哪些App不用VPN就能用

一个重要的事实：eSIM将你的数据通过中国网络基础设施路由。这意味着防火墙对eSIM流量的限制与对中国实体SIM卡完全相同。以下是能用和不能用的：

**不用VPN就能用：**
- Google Maps（网页版可以用；App部分被拦但通常能加载地图瓦片）
- Apple Maps
- 百度地图（强烈推荐在中国使用——在中国境内比Google Maps更精确）
- 微信、支付宝、滴滴出行、美团外卖
- Trip.com / 携程
- Airbnb、Booking.com
- Gmail（App和网页——稳定）
- Outlook/Hotmail
- WhatsApp（仅消息；通话质量可能不稳定）
- iMessage / FaceTime（好用）
- Instagram（加载得了但可能慢；信息流和故事更新间歇性）
- TikTok（用不了——区域锁定）
- YouTube、Facebook、Twitter/X、Google搜索、Google文档（全部被屏蔽——需要VPN）

**实际操作中的含义：** 你可以给家人发WhatsApp、查Gmail、用Apple Maps导航、用滴滴回酒店——全部不用VPN。但在等火车时刷不了Twitter，睡前看不了YouTube，除非装了VPN并保持开启。

许多旅客在出发前安装VPN，开启后叠加在eSIM连接上。这样获得无限制访问。eSIM提供数据管道；VPN提供访问权限。两者都需要。

## 数据测速——实际预期

在上海市中心使用Airalo（中国联通LTE）的实际测速结果，中午时段：

| 指标 | 数值 |
|------|------|
| 下载 | 12-35 Mbps（平均约22 Mbps） |
| 上传 | 3-12 Mbps（平均约7 Mbps） |
| 延迟（ping） | 至上海服务器40-80 ms；至美国西海岸180-250 ms |
| 视频播放 | 1080p YouTube（经VPN）流畅播放；4K偶有缓冲 |
| 视频通话 | FaceTime和WhatsApp视频清晰，网络拥堵时偶有马赛克 |
| 网页浏览 | 页面加载1-3秒；图片密集页面3-5秒 |

这些速度足够旅客所需的一切。不算家中光纤标准那么快，但很稳定。在地铁隧道中，连接短暂中断并重新连接。在密集城区高峰时段（下午5-7点），速度降至区间下限，因为网络承载更多用户。乡镇和偏远地区可能降到3-8 Mbps，但地图和消息仍可正常使用。

Holafly和Nomad使用同一中国联通网络基础设施，因此各供应商速度大致相当。差异在数据上限和定价上，不在网络性能上。

## 如何设置——步步详解

**出发前（在本地）：**

1. **检查手机的eSIM兼容性。** 前往 设置 → 关于手机，检查是否有EID或数字SIM。iPhone XS/XR及以后机型支持eSIM。多数三星Galaxy S20+、Google Pixel 4+及近期旗舰机支持。部分平价Android手机不支持。

2. **关闭本国运营商的国际漫游**以避免意外扣费。保持本国SIM活跃用于短信——仅关闭该线路的数据漫游。

3. **在供应商官网或App购买eSIM套餐。** 你将通过邮件收到二维码。

4. **安装eSIM配置文件。** 从另一台设备或打印件扫描二维码。你的手机会添加一个新蜂窝套餐。为清晰起见将其标注为"中国eSIM"。

5. **现在不要激活eSIM。** 在手机设置中保持禁用。只在落地时启用。

**抵达中国后：**

6. **连接机场WiFi**（大多数中国机场提供免费WiFi——寻找网络名称并按提示完成短信验证或护照号码登录）。

7. **在 设置 → 蜂窝网络/移动数据 中启用eSIM。** 将你的数据线路切换至中国eSIM。

8. **为eSIM线路打开数据漫游。** 这很重要——虽然你人身在中国，但eSIM技术上在中国网络上漫游。如果数据漫游关闭，eSIM不会连接。

9. **等待网络连接。** 这需要30-60秒。你会看到"中国联通"或"中国移动"作为网络名称和LTE或5G标识。

10. **测试连接。** 打开地图或网页。如果不能访问，检查数据漫游是否开启以及该eSIM是否设为活跃数据线路。

## 常见错误

- **忘记打开数据漫游。** eSIM在中国不能用的一号原因。旅行eSIM以中国运营商的漫游合作伙伴身份运行。如果该线路漫游关闭，就没有连接。
- **过早激活eSIM。** 如果你在离家前激活，套餐有效期开始倒数。只在到达时激活。
- **手机不支持eSIM。** 部分中国销售的手机和某些运营商锁定机型不支持eSIM。购买套餐前检查——退费不一定能拿到。
- **对VPN的期待。** eSIM提供标准互联网接入，受中国互联网法规约束。部分外国网站和App无论使用eSIM还是实体SIM都无法访问。eSIM的优势在于便利性，而不是无限制访问——网络基础设施是一样的。
- **使用错误的APN。** 如果连接速度慢或失败，检查eSIM供应商邮件中的APN设置。手动输入正确的APN有时能解决问题。

## 替代方案

- **在机场办理实体SIM卡**，如果你需要中国手机号用于App注册和本地通话。与eSIM结合作为备份数据源。
- **本国运营商的国际漫游。** 许多运营商提供中国数据套餐。价格更高（每天10-15美元），但无需任何设置，给你已知的稳定体验。
- **租用便携WiFi热点。** 部分机场和在线服务提供中国区域口袋WiFi租赁。为多台设备提供专用WiFi网络。费用与eSIM类似但需要取还流程。
