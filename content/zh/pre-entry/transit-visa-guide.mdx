中国为经中国前往第三目的地的旅客提供多项过境免签政策。适用哪一项取决于停留时长和入境城市。

## 三项过境免签选项——对比表

| 特征 | 24小时过境 | 144小时过境 | 240小时过境 |
|------|-----------|------------|------------|
| **时长** | 计划到达起24小时 | 144小时（6个日历日） | 240小时（10个日历日） |
| **适用国籍** | 所有国籍（持有有效联程机票） | 仅限54个国家 | 仅限54个国家 |
| **适用地点** | 全国所有开放口岸 | 20余个指定区域 | 仅限部分城市（重庆、西安、厦门） |
| **区域内移动** | 通常限于口岸区域；需临时入境许可方可离开 | 指定行政区域内自由移动 | 限于入境城市行政边界内 |
| **可否访问多城市** | 否——仅限于口岸城市及邻近区域 | 可以——在指定区域内（例如上海-苏州-杭州）| 否——仅限单一城市 |
| **出境口岸灵活度** | 须经同一区域（或同一口岸）的口岸出境 | 可从同一区域内任一指定口岸出境 | 须经入境城市的某一口岸出境 |
| **适用场景** | 短暂中转、当日联程 | 多日经停、商务出差、旅游 | 延长经停、短期商务 |
| **计时起点** | 计划到达时刻 | 到达后次日零时（晚上11点到达→次日零时开始计时=实际满满6天） | 到达后次日零时 |

### 路线示意图——过境如何运作

有效过境路线如下：

```
A国（出发地）→ 中国（经停城市）→ B国（目的地，须与A国不同）
```

有效路线示例：
- **伦敦 → 北京 → 东京**——英国经中国至日本。第三国不同。有效。
- **纽约 → 上海 → 香港**——美国经中国至香港特区。香港算第三地区。有效。
- **悉尼 → 广州 → 曼谷**——澳大利亚经中国至泰国。有效。
- **巴黎 → 成都 → 新加坡**——法国经中国至新加坡。有效。

无效路线示例：
- **伦敦 → 北京 → 伦敦**——返回出发国。非过境。无效。
- **首尔 → 上海 → 首尔**——往返。无效。
- **洛杉矶 → 上海 → 纽约**——出发和到达均为同一国家（美国）。无效。
- **东京 → 北京 → 大阪**——两者均在日本。无效。

规则很简单：你的目的地必须是与出发地不同的国家或地区。香港、澳门和台湾算作独立的第三地区，所以伦敦→上海→香港有效，尽管香港是中国的一部分（拥有独立的出入境系统）。

## 24小时过境（全国所有口岸）

最灵活的选择。**全国所有开放口岸**均可使用。如果你持有有效国际旅行证件和24小时内离境的确认联程机票，可以：
- 无需额外许可即在口岸限定区域内停留
- 如需离开口岸限定区域（例如入住酒店），可向边检申请临时入境许可

这基本覆盖所有人，不限国籍，只要你有联程机票。24小时从计划到达时间开始计算。

## 144小时过境——适用城市地图

对游客和商务旅客而言，这是最实用的选项。**54国公民**可在以下指定区域停留最长144小时（6天）：

**华东地区（上海-江苏-浙江区域）：**
- 出入境口岸：上海浦东（PVG）、上海虹桥（SHA）、南京禄口（NKG）、杭州萧山（HGH）
- 可在上海、苏州、杭州、南京、宁波以及江苏、浙江省内其他城市之间自由移动。
- 示例：飞抵上海PVG，乘高铁到杭州，再到苏州，然后从南京NKG出境。全部在同一区域内。

**华北地区（北京-天津-河北区域）：**
- 出入境口岸：北京首都（PEK）、北京大兴（PKX）、天津滨海（TSN）、石家庄正定（SJW）
- 允许在北京、天津及河北省城市之间移动。
- 去不了山西、内蒙古或其他邻省。

**华南地区（广东区域）：**
- 出入境口岸：广州白云（CAN）、深圳宝安（SZX）、揭阳潮汕（SWA）
- 允许在广东省全境移动。这意味着可以从广州入境，游览深圳，从任一城市出境。

**其他参与区域：**
- 辽宁（沈阳、大连）——双城市群
- 四川（成都）——城市区域
- 陕西（西安）——城市区域
- 厦门（福建）——城市区域
- 青岛（山东）——城市区域
- 武汉（湖北）——城市区域
- 昆明（云南）——城市区域

关键规则：你必须停留在已公布区域的行政边界之内。越界进入邻省就已离开合法过境区域。高铁和高速公路穿越这些边界时没有明显的实体检查站，但酒店登记系统、随机警察检查以及出境口岸的边检人员都能查出你去过哪里。

### 240小时过境（特定城市）

较新的选项，允许10天停留。目前重庆、西安和厦门口岸提供。适用同样的54国名单。必须在入境城市的行政边界内停留。适用于需要延长经停或单一城市短期商务出行的情况。

## 谁符合资格

54个适用国家包括：所有欧盟成员国、英国、美国、加拿大、澳大利亚、新西兰、日本、韩国、新加坡、马来西亚、泰国、俄罗斯、巴西、阿根廷、智利、墨西哥等。完整名单由国家移民管理局发布。

## 航空公司值机核查

你的航空公司在你到达中国之前就已经参与了过境流程。以下是具体情况：

**在出发机场**，值机人员核实你是否有权进入中国。当你出示没有中国签证的护照时，工作人员会检查：
1. 你的国籍在适用名单上（针对144/240小时过境），或者你持有24小时内前往第三国的有效联程机票（针对24小时过境）
2. 你的联程机票为已确认（非候补状态），且目的地为第三国
3. 你的出入境口岸与你计划使用的过境政策匹配

工作人员调出TIMATIC数据库（所有航空公司用于签证和入境要求核查的系统）并核实你的资格。如果TIMATIC标记问题——例如你的联程目的地与出发地相同——系统会显示警告，工作人员可能会拒绝登机。

**打印联程航班确认单。** 中国边检认可手机截图，但在出发机场值机柜台，工作人员通常想看纸质行程单，或至少手机上显示预订编号、航班号、日期和目的地的PDF。提前准备好可以避免延误。

**如果值机人员拒绝你登机**且你确信自己符合过境条件，请他们核查适用于你的国籍和口岸的具体过境政策。政策有变化，并非所有工作人员都掌握最新信息。必要时礼貌地请求主管复核。是否准予入境由中国边检说了算——航空公司值机核查是预筛，避免你在边境被遣返，航空公司对此负有责任。

## 过境印章长什么样

在中国边检，持过境免签政策入境时，边检人员在你的护照上盖一枚矩形入境章。与普通签证入境章不同的是，过境章有额外的中文文字表明你是一名过境旅客。

印章包含：
- 入境日期
- 入境口岸
- 类型："临时入境许可"或类似标识
- 手写或打印的允许停留时长（24H、144H或240H）
- 必须离境的截止日期

过境章看上去不像签证贴纸。它是一枚带有过境标识的标准出入境章。如果这个章与你在中国其他口岸收到的章不同，不必惊慌。

## 过境期间实际可以做什么

### 24小时过境
你有一天时间。实际可行的是：
- 离开机场，坐地铁到市中心，吃顿饭，看一两个景点，返回机场
- 订一家机场酒店睡觉（完全不游览）
- 开一个简短的商务会议，前提是会议地点距机场合理距离之内

现实做不到：走访多个城市、做完整的旅游行程或在口岸城市以外过夜。

### 144小时过境（6天）
算是一个正儿八经的迷你假期。你可以：
- 在指定区域内任意酒店办理入住
- 乘坐高铁、普通火车或国内航班在城市之间往返（全在区域内）
- 吃饭、游览景点、购物、开会——基本涵盖无签证游客或商务旅客的一切活动
- 使用打车软件、点外卖、用绑定外卡的App订火车票

你不能：
- 从不在指定区域的城市飞香港作为联程
- 去邻省看朋友
- 以任何理由延长停留超过144小时，除非是真实医疗紧急情况或不可抗力（航班取消、自然灾害）

### 240小时过境（10天）
与144小时类似，但限于单一城市。可以深度探索该城市、在同行政区域内周边城镇一日游、进行商务会议。多出的四天让节奏从容起来，不用赶景点。

## 容易出问题的关键规则

- **联程机票必须前往第三国/地区。** 伦敦飞北京再飞东京符合条件。伦敦飞北京再飞伦敦不符合。香港、澳门和台湾算第三地区。
- **必须留在行政区域内。** 每一项过境政策都绑定特定地理边界。离开这个区域就离开了合法过境区。
- **计时从到达后次日零时开始**（与免签入境相同）。如果你晚上11点到达，第一个"天"到次日零时才计数。
- **无法延长过境停留。** 错过了联程航班，必须离开过境区域直接到口岸出境——或者面对逾期后果。

## 常见错误

- **联程目的地搞错。** 从伦敦到北京再回伦敦的机票不算过境。目的地必须与出发地不同。
- **预订超出允许时间窗口的联程航班。** 如果联程航班在30小时后，而你只有24小时过境，则无法使用该政策。
- **混淆过境免签与普通免签政策。** 这是两项独立政策。30天免签入境要求你以中国为目的地；过境要求你只是途经。
- **并非区域内所有口岸都支持该政策。** 务必查看你所选入境口岸的具体政策公告。

## 步骤指南

1. **预订途经中国的航班**，持有过境时间窗口内前往第三国的确认联程机票。
2. **核实你的国籍**在你计划使用的具体过境政策的适用名单上。
3. **确认你的出入境口岸**均被列入该过境政策的名单。
4. **计划在指定区域内停留。** 不要在允许区域之外预订酒店或交通。
5. **在中国边检**告知官员你使用过境免签政策。出示护照和联程机票。
6. **全程保管好联程机票**——办理酒店入住或遇随机警察检查时可能需要出示。
7. **准时离境。** 提前到达机场赶联程航班，避免逾期风险。
