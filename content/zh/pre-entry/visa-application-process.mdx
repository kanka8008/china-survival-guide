如果你的国籍不在免签名单上，或者你打算停留超过30天、工作或学习，你需要签证。出行前申请。中国大陆没有落地签。整个流程通过两个官方在线平台进行。中介、代理、中间人都无法跳过这些步骤。

## 短期访客签证类型

你选一种签证类型。领事馆拒绝选错的申请。把你的实际活动对应到正确的类别。

- **L签证（旅游）。** 观光、短期探亲、休闲旅行。你需要往返机票、酒店预订和展示行程的基本行程单。30天单次入境L签证是首次申请人最常签发的短期签证。多次入境L签证（有效期6或12个月，每次停留30或60天）需要你护照上曾有中国签证。
- **M签证（商务贸易）。** 商业会议、贸易谈判、工厂考察、设备安装监督。你需要中国公司的邀请函。邀请函必须写明公司名称、访问目的、具体日期，并加盖公司正式圆形公章。你自己雇主的信函不算数。邀请函必须由中国接待方出具。
- **F签证（交流学术访问）。** 短期学术会议、文化交流、无偿讲座、非商业研究访问。需要中国机构的邀请函，盖公章要求同M签证。

长期类别：**Z**（工作）、**X1/X2**（学习）、**Q1/Q2**（家庭团聚）。短期签证不能在中国境内转换为长期类别。如果你持有L签证而中国公司给了你工作机会，你必须离开中国，在境外申请Z签证后重新入境。没有例外。

## 签证表格长什么样

打开领事服务平台 consular.mfa.gov.cn/VISA/，点击"开始我的申请"。表格以多页浏览器窗口加载。Adobe Flash死了。表格现在跑在HTML5上。Chrome和Edge能用。Safari有时在照片上传步骤出问题。Firefox如果关掉对网站的跟踪保护就能用。

表格有18个部分。每部分问什么：

- 第1部分：个人信息。名字严格按护照拼写。包括连字符、撇号和中间名。如果护照写"O'Reilly-Smith, James Patrick"，每个字符都要敲进去。
- 第2部分：护照信息。号码、发照国、签发日、到期日。必须和护照资料页逐字匹配。
- 第3部分：职业和雇主。写当前职位和公司名。"无业"可以接受但会被额外审查。领事馆职员可能问你怎么负担旅行费用。
- 第4部分：教育。最高学历和学校名称。学生填当前学校。
- 第5部分：家庭信息。父母、配偶和子女的名字。即使他们不和你一起出行也要填。
- 第6部分：旅行记录。列出过去5年每次去中国的行程和具体日期。如果你2019年去过12天，提供月份和年份。领事馆会拿内部记录核对。
- 第7部分：曾有的中国签证。类型、签发日、签发机构。如果之前的签证在过期护照里，上传那本旧护照的资料页和签证页扫描件。
- 第8部分：在华联系人。接待方或酒店的姓名、地址、电话。M签证和F签证必须与邀请函一致。
- 第9部分：紧急联系人。你本国能验证你身份的人。不是你的旅伴。
- 第10-14部分：行程信息。入境和离境日期、将访问的城市、住宿信息、谁支付旅行费用。保持一致。如果邀请函写5天而表格写14天，领事馆退回申请要求修改。
- 第15部分：健康和安全声明。如实回答每个问题。之前的签证拒签或入境拒绝在这里体现。领事馆已经知道。撒谎触发自动拒绝。
- 第16部分：照片上传。48mm x 33mm，白底，不戴眼镜，不戴头饰（宗教头饰除外），3个月内拍摄。系统自动拒绝有阴影、彩色背景或可见修图痕迹的照片。
- 第17部分：声明和签名。输入姓名作为数字签名。不要用花体签名。按表格上的姓名格式输入。
- 第18部分：审核和提交。点提交前逐字段阅读。提交后修改需要联系签证申请中心启动撤回流程，需要2到3个工作日。

表格每60秒自动保存。可以关掉后用"继续未完成申请"按钮回来。草稿30天后过期。设个日历提醒。

## 官方申请渠道

两个平台。没有别的。

1. **中国领事服务平台**（consular.mfa.gov.cn/VISA/）。三项功能：签证申请指南、开始我的申请、跟踪我的申请状态。如果你的国家没有专门的签证申请中心，或者你所在国家的中心走这个平台流转，就用这个。

2. **中国签证申请服务中心**（visaforchina.cn）。从下拉菜单选国家和城市。点"FOR VISA"。系统跳转到本地化门户。覆盖美国、英国、加拿大、澳大利亚、新西兰、欧盟成员国、日本、韩国和大多数有中国外交使团的国家。

选错地点导致申请失效。伦敦中心不能处理在曼彻斯特中心创建的申请。如果选错了城市，发邮件给原中心要求撤回，等确认后再重新开始。撤回需要2到3个工作日。损失的时间没有补偿。

## 分步指南

**第1步：选择正确的平台和地点。** 打开Chrome或Edge。对visaforchina.cn关闭弹窗拦截——照片上传和预约弹窗在拦截开启时不会加载。选择你当前合法居住的国家，除非国籍国和居住国一致。选择离你居住地最近的签证申请中心城市。

**第2步：完成在线申请表。** 填写每个带红色星号的字段。使用护照资料页上的准确拼写。连字符、撇号、空格、重音符号：全部复现。确实不适用的字段填"N/A"。空字段会让系统标记表格不完整。确认签证类型与实际目的相符。系统不会自动检查这个。这是你自己的活。

**第3步：上传支持性文件。** 上传：
- 护照资料页彩色扫描件（最低300 DPI，JPEG或PDF，照片和所有文字必须清晰可读）
- 符合规格的数码护照照片：48mm x 33mm，白底，禁止自拍，不戴眼镜，禁止修图
- M签证和F签证的邀请函：必须显示邀请方机构全名、地址、电话、访问目的和日期，以及一句该机构保证你遵守中国法律的声明。邀请函必须加盖机构正式圆形公章
- 合法居留证明（如果你在非国籍国申请）：居留许可、签证或写有本地址姓名的水电账单
- 机票预订：买一张可退票或请旅行社做预订保留。领事馆不要求已付款机票，但需要看到有姓名和航班号的预订

**第4步：等待初审。** 提交后系统检查完整性。在"跟踪我的申请状态"下追踪状态。三种常见状态：
- *审核中。* 领事馆职员在检查表格。无需操作。
- *等待提交护照。* 表格通过初审。预约到场办理。
- *退回修改。* 有问题。退回说明告诉你哪个字段需要修改。在说明指定的时间窗口内修改。

初审通常需要2到3个工作日。旺季（4月到9月，加春节前一周），预计5到7个工作日。

**第5步：预约到场办理。** 一旦出现"等待提交护照"，登录选时间。直接到场的会被门口拒绝。打印预约确认页。带着它。签证申请中心的保安会检查预约打印件和身份证件后才放你进门。

**第6步：到场提交并缴费。** 预约时：
- 保安在入口检查预约确认单和护照
- 取号等待。安静的周二早上等30分钟，公共假日后周一等2小时
- 叫号后走到柜台。职员对照原件和你上传的扫描件。任何不匹配就停办
- 职员给你拍照。同时在数码扫描仪上采集十指指纹
- 缴纳签证费。具体金额见下方
- 选择标准或加急服务
- 职员递给你一张领取凭条，上面有取证日期。丢了凭条不能取护照。立即拍照留存

**签证费用（2026年6月更新）。** 费用因国籍和签证类型而异。单次入境签证标准费用（美元）：

| 国籍 | L签证（旅游） | M/F签证（商务/交流） | 加急附加费 |
|---|---|---|---|
| 美国 | $185 | $185 | +$25 |
| 英国 | $125 | $125 | +$35 |
| 加拿大 | $100 | $100 | +$30 |
| 澳大利亚 | $65 | $65 | +$25 |
| 欧盟成员国 | $80 | $80 | +$30 |
| 日本 | $45 | $45 | +$20 |
| 韩国 | $40 | $40 | +$20 |
| 印度 | $90 | $60 | +$25 |
| 俄罗斯 | $55 | $55 | +$25 |

两次入境签证大约1.5倍单次费用。多次入境签证（6或12个月）大约2倍。签证费不退。如果申请被拒，钱不退。领事馆接受信用卡（Visa、Mastercard）、借记卡，部分地点接受当地货币现金。

**第7步：领取护照。** 按领取凭条上的日期回来。带凭条和带照片的身份证件。职员给你护照。翻到签证页。

## 签证贴纸长什么样

中国签证是一整页粘贴式贴纸。占护照一整页。贴纸底色浅绿或浅蓝，有长城暗纹水印图案。

看贴纸上的这些字段：

- **姓名。** 与护照完全一致的姓名。检查每个字符。这里少一个连字符或字母颠倒意味着签证无效。
- **护照号码。** 必须与护照资料页号码一致。领事馆偶尔会颠倒数字。柜台前核实。
- **签证类别。** 显示单字母：L、M、F、Z、X1、X2、Q1、Q2。确认与你申请的一致。你需要M签证却拿到L签证，需要重新申请。
- **入境次数。** 显示01（单次）、02（两次）或M（多次）。如果你订了两次入境的行程（飞上海、去东京、返回北京），单次入境签证让你回不来。离开柜台前修复。
- **每次停留天数。** 显示每次入境最大天数：030即30天、060即60天、090即90天。这是硬性上限。短期签证不能在中国境内延期。
- **签发日期。** 领事馆签发签证的日期。这不是有效期起始日。
- **Enter Before。** 你必须在此日期前用签证入境中国。超过这个日期，签证即使没用过也失效。大多数短期签证给90天有效期从签发日起算。部分领事馆只给30天有效窗口。订票前检查这个日期。
- **签发机构。** 显示城市和领事馆名称："北京""上海""伦敦"等。
- **照片。** 你提交的护照照片的激光打印黑白版。
- **二维码。** 左下角的方形矩阵条码。边检人员入境时扫描。不要折叠、刮擦或遮挡。

如果任何字段有误，立刻指出。职员有时可以当场修改小错误。重大错误（签证类别错、入境次数错）需要重新申请，重新缴费。不退款。

## 各领事馆处理时间

处理速度因地点而异。标准处理指收到护照后4到7个工作日。加急服务缩短至1到3个工作日。两个时限都不包含邮寄时间、周末、中国公共假日和申请所在国的公共假日。

2026年验证申请人的真实时间线：

| 领事馆/中心 | 标准（实际） | 加急（实际） |
|---|---|---|
| 伦敦 | 5-6个工作日 | 2-3个工作日 |
| 华盛顿DC | 6-8个工作日 | 2-4个工作日 |
| 纽约 | 6-8个工作日 | 2-4个工作日 |
| 洛杉矶 | 5-7个工作日 | 2-3个工作日 |
| 多伦多 | 5-6个工作日 | 2-3个工作日 |
| 悉尼 | 4-5个工作日 | 1-2个工作日 |
| 巴黎 | 4-5个工作日 | 1-2个工作日 |
| 柏林 | 5-6个工作日 | 2-3个工作日 |
| 东京 | 3-4个工作日 | 1个工作日 |
| 首尔 | 3-4个工作日 | 1个工作日 |
| 新加坡 | 3-4个工作日 | 1个工作日 |

美国领事馆速度较慢。在美国用标准服务申请，加上邮寄时间预留2整周。提前3周开始申请以吸收意外延误。护照从领事馆邮寄到你在其他州的地址，单程加3到5天。

**第8步：出行前检查。** 出发前，打开护照翻到签证贴纸。确认：有效期未过、入境类型与行程一致、允许停留天数覆盖整个行程、姓名拼写正确。把支持性文件——邀请函、酒店预订、回程票——装进随身行李。边检人员可能在入境柜台拿这些和你签证类型做交叉核对。

## 领事馆会问的问题

到场办理时，领事馆职员可能问表格之外的问题。最常问的以及你的回答应该传达什么信号。

**"你为什么来中国？"**

错误回答："我要帮朋友公司做点项目，顺便玩一下。"

正确回答："我4月3日至7日在上海参加上海贸易公司的商务会议。他们给了我邀请函。我4月8日返回。"

领事馆要一个单一明确的目的。旅游加商务是M签证，不是L签证。说你的主要活动。

**"谁支付你的旅行费用？"**

错误回答："不太确定。公司付一部分，剩下的我自己出。"

正确回答："我自己付机票和酒店。接待公司在商务会议期间承担餐饮和本地交通。我的银行流水在支持性文件里。"

领事馆检查你是否有足够的钱。如果别人付钱，那个人需要出现在文件中。

**"过去5年你去过哪里？"**

职员在拿你的回答和表格上的旅行历史部分做对比。任何不一致引发追问。照着表格回答。不要加忘了列的行程。

**"你在哪里工作，什么职位？"**

领事馆用你的申请表核实就业情况。学生和无业申请人应清楚说明当前状态："我是多伦多大学大三学生"或"我目前无业，用储蓄支付这次旅行。"

**"你去过中国吗？"**

错误回答："去过，几次吧，具体日期记不清了。"

正确回答："去过，2022年3月持L签证去了10天。之前签证在过期护照里。我上传了扫描件。"

领事馆系统里已经看到你的中国旅行史。他们在测试你的诚实度。如果忘了列某次行程，现在告诉他们。这比他们自己发现然后认定你故意隐瞒好看。

**"你在中国有亲属吗？"**

如果有，说他们是谁、住哪里、和你的关系。领事馆在检查潜在的逾期滞留风险。有亲属不是取消资格的理由，但隐瞒看起来可疑。

## 常见陷阱

- **签证类型选错。** 实际目的是商务却申请L签证（旅游）。支持性文件里的邀请函说"商务会议"。表格上签证类型选"旅游"。领事馆退回。选邀请函描述的活动类型对应的签证类型。
- **邀请函问题。** 最高拒签原因：缺公章、日期错误、接待公司名称与营业执照不一致、或发函公司非中国注册企业。让接待方在提交前发盖章邀请函扫描件给你。自己检查公章。正规圆形公章有公司中文全名环绕一颗五角星。
- **护照照片被拒。** 自拍、戴眼镜、彩色背景、修图磨皮、耳后阴影。去专业照相馆。告诉他们需要中国签证照片。他们知道48mm x 33mm规格。费用15到25美元一套4张。每一分都值。
- **空白页不够。** 护照需要一整页完全空白的签证页。不是加注页，不是签注页。什么都没有的页面。如果护照只剩半页空白或有一页只有一枚入境章，不算。申请前换护照。
- **曾有过逾期记录。** 中国边检数据库记录每个外国人的出入境。任何逾期都出现在你的档案里。3年前逾期2天照样显示。领事馆可能对过去轻微逾期签发签证，但会问。准备好说明。反复逾期会阻断未来签证审批。

## 处理时间和费用

标准处理：4到7个工作日。加急服务：1到3个工作日。这些是具体领事馆地点的工作日，不是日历日，也不是中国的工作日。如果在伦敦复活节周末期间申请，时钟从周五到周一暂停。如果在中国国庆节（10月1-7日）期间申请，领事馆关闭整整一周。

没有当天出签服务。加急附加费买的是处理队列中更短的排队位置，不是即时处理。按两个缓冲期规划申请时间线：1周准备（填表、收集文件、拍照）、1周初审、1周标准处理、1周缓冲。总计：从开始到护照到手4周。出行前4周开始。
